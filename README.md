# 42-fdf
Infographie project map relief.

Installation
---
Include Minilibix Library and launch:

    Make

Usage
---
Execute program with path map file

    ./fdf map_name.fdf

Control
---
#####Translation#####
  * Y: Key: UP, DOWN
  * X: Key: LEFT, RIGHT

#####Rotation#####
  * X: Keypad: 1, 4 
  * Y: Keypad: 2, 5
  * Z: Keypad: 3, 6

#####Zoom#####
  * IN: Keypad: +
  * OUT: Keypad: -
